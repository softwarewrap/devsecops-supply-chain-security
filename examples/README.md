# Examples

This directory contains small example projects demonstrating how to integrate the DevSecOps security pipeline into real repositories.

Planned examples include:

- python-project
- container-project
- minimal GitHub Actions integration

These examples will demonstrate how to:

• run the CI security pipeline
• generate SBOMs
• upload SBOMs to Dependency-Track
• import scan results into DefectDojo

If you contribute an example integration, please ensure it is small, reproducible, and easy for readers to run locally.
